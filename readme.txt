Transform your content into a stunning Matrix-style typing animation with customizable effects.

== Description ==

Bring the iconic Matrix digital rain effect to your WordPress site! This plugin replaces standard text with:

- Realistic character-by-character typing animation
- Customizable "green code" color scheme
- Adjustable typing speed and effects
- Optional falling symbols background
- Smart disabling on selected pages

Perfect for:
✓ Tech blogs
✓ Cyberpunk-themed sites
✓ Portfolio showcases
✓ Gaming websites
✓ Special announcements

= Key Features =
* Multiple cursor styles (blinking, static, underline)
* Background effects customization
* Selective page targeting
* Mobile-responsive design
* Lightweight (under 50KB)
* WPML/RTL compatible

== Installation ==

1. Upload `matrix-effect` folder to `/wp-content/plugins/`
2. Activate through 'Plugins' menu
3. Configure in *Settings → Matrix Effect*
4. Add shortcode `[matrix-effect]` to posts/pages (optional)

For advanced placement:
`<?php do_action('matrix_effect_display'); ?>`

== Frequently Asked Questions ==

= How to change the text color? =
Go to Settings → Matrix Effect → Text Color

= Can I use different colors than green? =
Yes! Any HEX color is supported in settings.

= How to disable on mobile? =
Add this to your theme's CSS:
`.mobile-device #matrixEffectContainer { display:none; }`

= Does it work with page builders? =
Yes, compatible with Elementor, Gutenberg and others.
